Welcome to OpenCPX!
==============

Opencpx is a site administration web interface, as well as a mail and file manager for end users. 

Prerequisites
--------------

perl
apache
mod_perl
sendmail/postfix
vsftpd/proftpd
dovecot

yum repo for latest rpm
--------------
